# 🎵 Jam App

A collaborative social platform for musicians to connect, collaborate, and create music together.

---

## 📖 Overview

**Jam App** is a Full Stack Web Application developed as my **Final Year BCA Project**. It is designed to bring musicians together by providing a platform where they can connect with like-minded artists, showcase their musical skills, collaborate on projects, and communicate in real time.

The application offers an engaging user experience with modern UI, secure authentication, and real-time communication features.

---

## ✨ Features

- 👤 **User Profiles**
  - Create and manage personalized musician profiles.
  - Showcase instruments, skills, genres, and interests.

- 🔍 **Find Musicians**
  - Search musicians by instrument, genre, city, or skill level.
  - Discover collaborators based on shared interests.

- 💬 **Real-Time Chat**
  - Send and receive instant messages.
  - Communicate seamlessly with fellow musicians.

- 🎼 **Jam Sessions**
  - Create or join virtual jam sessions.
  - Collaborate with musicians online.

- 📚 **Learning Corner**
  - Share tutorials, lessons, tips, and musical resources.
  - Learn from experienced musicians.

- ❤️ **Community Features**
  - Connect with other musicians.
  - Build your musical network.

- 🔐 **Secure Authentication**
  - User Registration & Login
  - JWT Authentication
  - Protected Routes

- 📱 **Responsive Design**
  - Optimized for Desktop, Tablet, and Mobile devices.

## 🚀 Getting Started

### Navigate to the Project

```bash
cd Jam-App
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Run the Project

Frontend

```bash
npm start
```

Backend

```bash
npm run dev
```
